Installation Guide

Follow the steps below to install and use Mongolian fonts and input tools.

Windows System

  1. Download the required font from the Font Library.
  2. Right-click on the font file and select "Install."
  3. Set up the Mongolian input method in "Language Settings."

Mac System

  1. Download the required font from the Font Library.
  2. Double-click on the font file and click "Install Font."
  3. Enable Mongolian input in "System Preferences > Keyboard."

Mobile Devices

  1. Download a Mongolian keyboard app from your app store.
  2. Install and activate the keyboard in your device settings.
  3. Select Mongolian as the input method.
